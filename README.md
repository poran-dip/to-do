# To Do

A PWA To-Do List app built with vanilla HTML, CSS and JavaScript, using localStorage for persistence.

Add, edit and delete tasks, and mark them as done.

One of my earliest projects, built in early 2025. Includes custom branding with icons, favicons, and a proper PWA manifest — was obsessed with being thorough with my projects even back then.

Check it out on [poran-dip.github.io/to-do](https://poran-dip.github.io/to-do/).
